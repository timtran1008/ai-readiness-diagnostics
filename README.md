# AI Readiness Diagnostics

Most teams don't have an AI problem first. They have a workflow, environment, and decision-quality problem.

This toolkit helps leaders, team leads, and internal AI champions figure out what to fix before spending money or time on AI.

## Live Assessments

Try the interactive assessments directly in your browser:

- [Individual AI Proficiency Assessment](https://muriel1008.github.io/ai-readiness-diagnostics/individual-assessment.html) — Adaptive assessment using the TOE framework (Tool fluency, Orchestration, Evaluation)
- [Organization AI Maturity Assessment](https://muriel1008.github.io/ai-readiness-diagnostics/organization-assessment.html) — Team-level diagnostic using the GIE framework (Governance, Infrastructure, Efficiency)

## How to use the prompts

1. Copy the full prompt text
2. Paste it into any AI assistant
3. Answer the questions one at a time — the AI will guide you through
4. Get a scored, actionable output

The prompts push for specifics, use scoring anchors, apply explicit decision rules, and flag weak evidence — not vague observations.

## Framework

All diagnostics share the same [operating logic](tim-diagnostic-framework.md):

- Ask one question at a time
- Push for specifics — if the answer is vague, say so
- Separate technical, workflow, leadership, and domain/regulatory layers
- Score with anchors, not gut feel
- Apply decision rules to reduce subjectivity
- End with a decision, a first-action list, and what NOT to do
